Instrucciones:

  1. Genera un token de GitHub en github.com/settings/tokens
  2. Asegúrate de activar el permiso "Models" en tu token
  3. Pega el token arriba y haz clic en "Guardar Token"
  4. Selecciona el modelo que desees utilizar
  5. ¡Empieza a chatear!

Nota: GitHub limita el uso gratuito a ~50 consultas diarias para GPT-4o.
